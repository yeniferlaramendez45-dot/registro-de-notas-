Registro de Notas

Este proyecto es un programa en C# que permite:

- Registrar estudiantes
- Registrar materias
- Ingresar notas
- Mostrar notas en tabla
- Calcular y mostrar promedio por estudiante

Cómo usarlo

1. Abrir el programa.
2. Ingresar la cantidad de estudiantes y materias.
3. Usar el menú para registrar estudiantes, materias y notas.
4. Visualizar notas y promedios.
